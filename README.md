                               Sobre o Projeto



Tecnologias e conceitos aplicados
HTML5

Utilizado para estruturar toda a aplicação, organizando elementos como navegação, formulário, campos de entrada e área de exibição dos dados.

CSS3

Responsável pela identidade visual do projeto. Foram aplicados conceitos de:

estilização moderna;
organização de layout;
responsividade inicial;
efeitos visuais;
hierarquia visual;
personalização de componentes;
experiência do usuário.
JavaScript

Responsável pela lógica da aplicação e interação com a interface.

Durante o desenvolvimento foram aplicados conceitos como:

captura de elementos através do DOM;
manipulação dinâmica de conteúdo;
eventos;
atualização em tempo real da interface;
criação de elementos dinamicamente.
Consumo de API e Requisições Assíncronas

Foi utilizada uma API pública para obtenção das informações relacionadas às coordenadas geográficas da cidade informada.

Conceitos aplicados:

requisições assíncronas;
uso do método fetch;
tratamento das respostas da API;
interpretação dos dados retornados.
Validação e Tratamento de Erros

Também foi implementada uma etapa de validação para melhorar a experiência do usuário:

verificação dos campos obrigatórios;
mensagens de erro personalizadas;
prevenção de envio de dados inválidos;
tratamento de possíveis falhas durante as requisições.
Objetivo do projeto

Mais do que construir uma interface visual, este projeto teve como objetivo praticar conceitos fundamentais do desenvolvimento Front-End e compreender como diferentes tecnologias trabalham juntas para gerar uma aplicação funcional.
